# Assets Directory

This directory contains all media assets for the website.

## Structure

```
assets/
├── favicon.png          # Site favicon
├── images/              # Image files (PNG, JPG, WEBP, SVG)
│   ├── placeholder-cover.webp
│   ├── placeholder-comparison.png
│   ├── placeholder-relighting.webp
│   ├── placeholder-overview.jpg
│   ├── placeholder-feature1.png
│   └── placeholder-feature2.png
└── videos/              # Video files (MP4)
    ├── placeholder-bg.mp4
    └── placeholder-video.mp4
```

## Placeholder Files

All files with `placeholder-` prefix are placeholders. Replace them with your actual assets:

- **placeholder-bg.mp4** - Hero section background video
- **placeholder-video.mp4** - Main intro video
- **placeholder-cover.webp** - Video cover/thumbnail image
- **placeholder-comparison.png** - Comparison chart image
- **placeholder-relighting.webp** - Relighting example image
- **placeholder-overview.jpg** - Pipeline overview diagram
- **placeholder-feature1.png** - Feature illustration 1
- **placeholder-feature2.png** - Feature illustration 2

## Adding Your Assets

1. Replace all `placeholder-*` files with your actual assets
2. Keep the same file names or update the paths in `index.html`
3. Recommended file formats:
   - Images: WebP (preferred), PNG, JPG
   - Videos: MP4 (H.264 codec recommended)
   - Keep file sizes optimized for web
